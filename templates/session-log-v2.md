# Session Log v2 — 含量化字段

> 相比 v1（纯文本流水账），v2 在每个阶段结束时增加了量化字段。
> 这些字段是 M2（行为量化）和 M3（效果验证）的数据来源。

## 元数据

| 字段 | 说明 | 填写者 |
|------|------|:--:|
| 项目名称 | | 开发者 |
| 开始日期 | YYYY-MM-DD | 自动 |
| 项目分级 | 面试项目(最高深度) / 生产项目(高深度) / 测试用例(最低深度) | 开发者 |
